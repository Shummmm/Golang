Проект на Golang для управления библиотекой

Реализованы структуры User, Book, Library, которые имплиментриуют интерфейс Identifiable

Функции:
1) Добавление книг и пользователей
2) Возможность взять и вернуть книгу, с автоматическим обновлением статуса доступности книги
3) Поиск книг и пользователей по ID
4) Список всех книг и всех доступных в данный момент
